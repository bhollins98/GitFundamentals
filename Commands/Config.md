# git config

When first setting up git, you'll need to work with the git config commands in order to set up your identiy.

For example, your identity can be set up withe the commands such as:

```
git config --global user.name "Benjamin Hollins"
git config --global user.email indyben@sbcglobal.net
```

For more information, view the [first time git setup] (https://git-scm.com/book/en/v2/Getting-Started-First-Time-Git-Setup) documentation
